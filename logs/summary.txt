## Common Issues Found and Resolved

### Structural Issues
- Missing NAMASTE declaration files (0=ocfl_object_1.1)
- Missing or incorrect inventory.json files
- Invalid version directory naming (zero-padding, incorrect format)
- Unexpected files in object root or version directories
- Missing version directories in sequence

### Metadata Issues
- Incorrect digest algorithms (MD5 instead of SHA512, algorithm mismatches)
- Missing or malformed 'head' field in inventory
- Invalid timestamp formats (missing timezone, incorrect ISO 8601 format)
- Invalid object IDs (non-URI format)
- User address not in URI format

### Content and Manifest Issues
- Digest/checksum mismatches between inventory and actual files
- Missing content files referenced in manifest
- Duplicate or conflicting path mappings
- Content in wrong directory locations
- Missing manifest entries for existing files

### Inventory Consistency Issues
- Inventory checksum (SHA512 sidecar) mismatches
- Inconsistent inventories between root and version directories
- Version state mismatches between different inventory files
- Path conflicts and invalid path formats
